A simple search view to help you to search easily Booking Order by Product Serial Number, Product Name, Customer and Order Number. You can also see List of Products, Delivery Order and Invoices in Booking Order Calendar View.

Product Configuration

Flexible Product for Booking Order


In the product, if "Can be Booked" is true then "Default Preparation Days", "Default Buffer Days", "Serial Number" are required and tracking should be none. You can use only those products which have the same configuration.

Validate Booking Order

After "Validate" the Booking Order it will create Delivery Order automatically in draft state and Booking Order state would change from draft to out, If you "Cancel" the Booking Order it will come in cancel state

In Booking Order Lines state represents the state of respective Delivery Order.

Sale Product

Create Invoice Or No Invoice

"Create Invoice" will create the Invoice and process the Delivery Order for selected products from wizard.

"No Invoice" will Cancel the Draft Invoice, If not in Draft it will Refund. For Delivery Order, It will Cancel the Draft Delivery Order, If not in Draft it will Return.

By Clicking on Delivery Order Smart Button you can go in delivery order and Delivery order have Tree, Kanban and Calendar View. Also Invoice Smart Button will be visible once Invoice is created.

Sold Booking Order

"Sold" will create the Invoice and process the Delivery Order to Done for products which is in Booking Order.
